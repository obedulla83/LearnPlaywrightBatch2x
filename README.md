# LearnPlaywrightBatch2x

This repository contains learning materials and code examples for Playwright, a Node.js library for automating web browsers. It covers JavaScript basics, concepts, and Playwright-specific topics.

## Repository Structure

```
chapter_01_Basics/
├── 01_Basics.js
├── 02_JS.js
├── 03_JS_Commands.js
└── 04_HotCode.js

chapter_02_Javascript_Concepts/
├── 05_JS_Basics.js
└── 07_JS_Identifiers.js

chapter_03_Identifier_Literals/
├── 06_Identifier_Rules.js
├── 07_Identifier_Part2.js
├── 08_Comments.js
├── js_Identifiers_rules.js
├── VS_Code_Keybaord_shortcut_Windows.md
└── VS_Code_Keyboard_shortcut_mac.md
```

## Chapters Overview

### Chapter 1: Basics
- Introduction to basic JavaScript concepts
- Fundamental programming constructs
- Hot reloading and code execution

### Chapter 2: JavaScript Concepts
- Advanced JavaScript features
- Identifier rules and naming conventions
- Best practices for variable naming

### Chapter 3: Identifier Literals
- Detailed rules for JavaScript identifiers
- Different naming conventions (camelCase, snake_case, PascalCase, etc.)
- Hungarian notation examples
- VS Code keyboard shortcuts for Windows and Mac

## Prerequisites

- Node.js (version 14 or higher)
- npm or yarn
- Git

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/obedulla83/LearnPlaywrightBatch2x.git
   ```

2. Navigate to the project directory:
   ```bash
   cd LearnPlaywrightBatch2x
   ```

3. Install dependencies (if any):
   ```bash
   npm install
   ```

## Usage

Each chapter contains JavaScript files with examples and explanations. You can run the files using Node.js:

```bash
node chapter_01_Basics/01_Basics.js
```
