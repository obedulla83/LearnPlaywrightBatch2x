# LearnPlaywrightBatch2x

A comprehensive JavaScript learning resource covering fundamentals, concepts, and best practices for web automation with Playwright.

## Project Structure

### Chapter 01: Basics
- **01_Basics.js** - Introduction to basic JavaScript concepts
- **02_JS.js** - JavaScript fundamentals
- **03_JS_Commands.js** - Essential JavaScript commands
- **04_HotCode.js** - Hot code reloading and best practices

### Chapter 02: Javascript Concepts
- **05_JS_Basics.js** - Core JavaScript basics

### Chapter 03: Identifier & Literals
- **06_Identifier_Rules.js** - Rules for naming identifiers
- **07_Identifier_Part2.js** - Advanced identifier concepts
- **08_Comments.js** - Comment syntax and best practices
- **js_Identifiers_rules.js** - Comprehensive identifier rules reference
- **VS_Code_Keybaord_shortcut_Windows.md** - VS Code shortcuts for Windows
- **VS_Code_Keyboard_shortcut_mac.md** - VS Code shortcuts for macOS

### Chapter 04: Javascript Concepts (Advanced)
- **09_var_let_const.js** - Variable declaration differences
- **10_functions.js** - Function declarations and usage
- **11_var_explained.js** - Understanding `var` keyword
- **12_let_people_love.js** - Understanding `let` keyword
- **13_const_explained.js** - Understanding `const` keyword
- **14_var_functionscope.js** - Function scope with `var`
- **15_let_scope.js** - Block scope with `let`
- **16_Hoisting.js** - Variable hoisting concepts
- **17_Hoisting_fn.js** - Function hoisting concepts

## Getting Started

1. Clone this repository
2. Open files in your preferred code editor (VS Code recommended)
3. Follow the chapters in order for structured learning
4. Run JavaScript files using Node.js: `node filename.js`

- Variable declarations (`var`, `let`, `const`)
- Function declarations and expressions
- Scope (global, function, block)

- Comments and documentation
